🚀 CodeRefine-AI

Generative AI-Powered Code Review & Optimization Engine

CodeRefine-AI is an intelligent AI-powered platform that analyzes source code, detects bugs, identifies performance issues, checks security vulnerabilities, and generates optimized rewritten code.

The system uses Groq powered Llama-3.3-70B large language model to perform real-time code analysis and rewriting. This helps developers improve code quality, maintain coding standards, and accelerate development workflows.

---

📌 Project Overview

Traditional code reviews require manual effort and time.
CodeRefine-AI automates this process using Generative AI.

The platform reviews code and provides:

• Bug Detection
• Performance Optimization
• Security Issue Detection
• Best Practice Suggestions
• Clean Rewritten Code

It converts raw code into production-ready optimized code with explanations.

🧠 How It Works

1. User pastes source code into the web interface
2. Backend API sends request to Groq LLM (Llama-3.3-70B)
3. AI analyzes the code for bugs, security issues and performance problems
4. System generates a detailed review report
5. Optimized rewritten code is produced

This allows developers to get instant intelligent code feedback.

⚙ Installation & Setup

Clone the repository

git clone https://github.com/anshulagg2730/CodeRefine-AI

Go to project folder

cd CodeRefine-AI

Go to backend folder

cd backend

Activate virtual environment

venv\Scripts\activate

Install dependencies

pip install -r requirements.txt

Add your Groq API key inside ".env"

GROQ_API_KEY=your_api_key_here

Run the application

uvicorn main:app --reload

Open in browser

http://localhost:8000
